### Client DataBase is a Unity tool for reading planner-authored Excel tables and generating matching code plus [ScriptableObject](https://unity3d.com/cn/learn/tutorials/modules/beginner/live-training-archive/scriptable-objects) assets automatically.



### Features

-   Unity editor tooling.
-   One-click resource generation.
-   Configuration profiles.
-   Data category management.
-   Open source code.
