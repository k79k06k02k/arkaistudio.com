**Dodge Jam / Subtitle Rhythm Action**
A solo entry for the 2017 KUSO Game Jam, built with Unity-Chan assets in about 12 hours and released on itch.io.

-   Designed the core game concept.
-   Built the game flow.
-   Implemented character controls.
-   Implemented subtitle-driven level design and timing.

2017 KUSO Game Jam: [https://itch.io/jam/kuso-game-jam-2017](https://itch.io/jam/kuso-game-jam-2017)
