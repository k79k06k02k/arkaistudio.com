**MEandMine Emotion Lab for Kids / EdTech game platform**

-   Built GenAI features for educational products with Gemini, OpenAI, and Vertex AI RAG, including character generation, story videos, and guidance chatbot workflows.
-   Ported an iOS Unity game to WebGL and rebuilt the downloadable content flow with Addressables, reducing the initial bootstrap package from roughly 700 MB to 15 MB.
-   Kept runtime memory under 150 MB at 30 FPS targets on Chromebooks and iPad Safari.
-   Simplified core gameplay architecture by roughly 50% through refactoring, improving code quality and maintainability.
-   Built Python backend APIs, analytics integrations, AI workflow services, Jenkins CI/CD, TestFlight release flow, and production monitoring with n8n, Elastic, Grafana, and Slack alerts.
