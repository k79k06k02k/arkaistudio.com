**Myth Saga / Card Game**

-   Managed a programming team of fewer than 10 engineers, covering schedule control, production process optimization, cross-discipline communication, technical decisions, mentoring, and low-level development.
-   Built a shared module architecture with Unity Package Manager and an npm server, including networking foundations, Excel data loading, asset management with Addressables and Android App Bundles, and a UI framework.
-   Planned Git training and the development workflow.
-   Implemented automation for asset validation, packaging, automated testing, and performance analysis, then shared the workflow internally.
-   Improved engineering workflow, code style, and code review standards.
-   Owned project quality control, performance cost analysis, reporting, and optimization.
-   Wrote and maintained technical documentation for sync meetings, records, and handoff.
